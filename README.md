
# FamilyPoints (Offline Web-App ohne Installation)

Diese Version läuft **ohne Installation** direkt im Browser (Chrome/Edge/Firefox/Safari).

## Nutzung
1. ZIP entpacken.
2. `index.html` doppelklicken (öffnet sich im Standard-Browser).
3. Die Daten werden lokal per `localStorage` gespeichert.

## Features
- Kind-Ansicht: Punkteanzeige, Aufgabenliste, Belohnungen.
- Eltern-Ansicht: PIN-Schutz, Aufgaben/Belohnungen verwalten, Punkte anpassen.
- Verlauf mit Zeitstempel.
- Alles offline, kein Server nötig.

## Hinweise
- Läuft auch auf Smartphone: Per USB kopieren und im mobilen Browser öffnen.
- Optional später: Hosting über GitHub Pages (nur Dateien hochladen). Dann kann man es wie eine App speichern (PWA).
